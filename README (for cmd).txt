JDBC — платформенно-независимый стандарт взаимодействия Java-приложений с различными СУБД.
JDBC реализован в виде пакета java. sql, входящего в состав Java SE (Standart Edition).

То есть выбираем БД (DerbyDB, Oracle итд), скачиваем нужный для неё драйвер,
   а код по взаимодействию остается тем же (используется библиотека java.sql для JDBC)

1) chcp 1251
2) startNetworkServer -> port 1527
3) open new cmd window
4) chcp 1251
5) ij (утилита для создания БД)
6) connect 'jdbc:derby://localhost:1527/Lesson22;create=true';
7) приконектились к существующей БД, если её не было, то создали пустую (далее в ней есть TABLE)
8) работа с БД с помощью SQL команд через утилиту ij в командной строке
9) в проект надо добавить external jar derbyclient.jar (это и есть драйвер)
10)* Для эффективного использования БД надо писать код с использованием DataSource
